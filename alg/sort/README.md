各种排序算法
===

* [堆排序](heapSort.cpp)

* 希尔排序
	分为几轮排序,每轮排序以一个步长来排序元素,并且通常采用选择排序.轮之间的步长递减,直到为1,排序工作蜕变为插入排序.

* 桶排序
	把数据分成几个区间,每个区间称为一个筒.先把筒内的数据排序好,再按照区间范围顺序,便利所有筒即可.这需要筒子较多,并且数据分布均匀才能有很好的效果,大概是`O(n+m*n/m*log(n/m))=O(n+nlog(n/m))`,可见n和m相等,并且筒内采用快排时,时间复杂度为O(n).但是,筒子越多,占用空间也越多.

* 基数排序
	如果数据可以分成好几位,那么可以根据位来排序.先排个位,然后十位,直到最高位.每次排序先按某个位排好,下次排序继续精确排序结果.我认为复杂度取决与系数K,即是数字位的个数.时间复杂度为O(klog(n))

* 计数排序
	需要开辟一个大数组a,元素a[i]代表元素i出现的个数.最后只要遍历数组a即可.


可以看到,很多排序都是利用这个思想:处理局部已排序的数组,要比处理完全没有被排序的数组快.
